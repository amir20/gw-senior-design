System Requirements

The following are the hardware requirements for Voice Launch. Please note that speakers are not required but only recommended.

    • Microphone
    • Speakers (Not required but recommended)
    • At least 800 MHz Processor
    • At least 256 MB RAM
    • Keyboard and Mouse
    • Enough disk space to save the WAV files

The following are software requirements for Voice Launch. Voice Launch requires Windows XP or later with Java Virtual Machine 1.5 or higher installed.

    • Windows XP or later
    • Java Virtual Machine 1.5 or later
